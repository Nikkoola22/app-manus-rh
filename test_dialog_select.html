Instructions de Test
  1. Testez les menus déroulants : Cliquez sur chaque menu déroulant et vérifiez que les options sont visibles et sélectionnables.
  2. Vérifiez la transparence : Les menus ne doivent pas être transparents ou difficiles à lire.
  3. Testez la sélection : Sélectionnez différentes options et vérifiez qu'elles s'affichent correctement.
  4. Testez le formulaire : Remplissez le formulaire complet et soumettez-le.
  5. Comparez avec l'application : Les menus dans l'application doivent avoir le même comportement.
